# 需要删除的短语

## 开场铺垫

删除这些清嗓句。直接说内容。

- “值得注意的是”
- “需要指出的是”
- “不可否认的是”
- “众所周知”
- “我们不难发现”
- “从某种意义上说”
- “在当今快节奏的时代”
- “在这个信息爆炸的时代”
- “随着社会的发展”
- “在当前复杂多变的环境下”
- “事实上”
- “说到底”
- “归根结底”
- “毋庸置疑”
- “必须承认”
- “让我们来看看”
- “接下来，我将”
- “本文将”
- “本文旨在”

凡是先宣布“我要说什么”的句子，都先删掉，再看剩下的内容能不能直接开头。

## 强调拐杖

这些词制造力度，却不增加信息。删掉或换成事实。

- “这很重要”
- “这才是关键”
- “关键在于”
- “请记住”
- “不夸张地说”
- “换句话说”
- “说白了”
- “更重要的是”
- “这背后真正的问题是”
- “这正是 X 的意义”
- “这件事值得每个人思考”

“换句话说”只有在后一句真的更清楚时才保留。多数情况下，它只是重复。

## 内容营销和商业套话

换成普通话和具体动作。

| 避免 | 改成 |
|------|------|
| 赋能 | 帮助、让、给 |
| 闭环 | 完成、做完、接上 |
| 抓手 | 方法、动作、入口 |
| 落地 | 做成、执行、上线 |
| 沉淀 | 记录、留下、形成 |
| 提效 | 节省时间、减少步骤 |
| 对齐 | 确认一致、说清楚 |
| 场景 | 情况、用法、时机 |
| 生态 | 伙伴、产品组合 |
| 链路 | 流程、步骤 |
| 打法 | 做法 |
| 壁垒 | 优势、门槛 |
| 价值 | 结果、收益、影响 |
| 洞察 | 发现、判断 |
| 引爆 | 带来、促成、推动 |

不要只把套话换成另一个套话。能写数字、动作、对象，就写出来。

## 程度词和软化词

删除没有事实功能的程度词、软化词和套话副词。保留承担事实边界的限定词，比如法律、医学、金融判断里的“可能”。

常见问题词：

- “其实”
- “只是”
- “非常”
- “特别”
- “比较”
- “相对”
- “某种程度上”
- “可以说”
- “可能”
- “或许”
- “基本上”
- “显然”
- “真正”
- “深度”
- “高度”
- “持续”
- “不断”
- “全面”
- “有效”
- “显著”
- “充分”
- “积极”
- “重要”
- “核心”

如果删掉一个词后意思没变，就删。

## 元评论

删掉文本对自己的解释。文章应该往前走，不要介绍自己怎么走。

- “下面我们将”
- “本节将”
- “这篇文章旨在”
- “本文试图回答”
- “希望能给你启发”
- “欢迎在评论区”
- “如果你也”
- “这不是鸡汤”
- “你一定要看到最后”
- “让我们一步步拆解”
- “接下来进入正题”

## 虚假真诚

这些表达制造亲近感，却通常没有内容。

- “说实话”
- “老实说”
- “我必须坦白”
- “我知道这听起来”
- “相信我”
- “真的”
- “我一直认为”
- “你可能已经意识到了”

## 讲道理不举例

这些句子在宣布难度、意义或正确性，却没有展示任何证据。

- “这是一件很难的事”
- “这体现了真正的领导力”
- “这正是产品思维的体现”
- “这背后有深层原因”
- “这说明问题并不简单”
- “这才是长期主义”
- “这就是增长的本质”

写出具体人、具体冲突、具体代价。不要用标签替代判断。

## 空泛判断

下面这些判断没有对象、尺度或证据。删掉，或换成具体影响。

- “意义重大”
- “影响深远”
- “值得深思”
- “潜力巨大”
- “前景广阔”
- “问题复杂”
- “原因是多方面的”
- “挑战依然存在”
- “具有重要价值”
- “带来新的可能”
- “形成有效支撑”
- “提供有力保障”

如果一句话说某事重要，却没有说明重要给谁、重要在哪里、造成什么变化，这句话还没写完。
